OpenData API


GET data/from/{city}

Get all data from a specific city

Requirements:

  • City - The city you want to fetch data from

GET data/{data point}/from/{city}

Get a specific type of data from a specific city

Requirements:

  • Data point - What kind of data you want to get
  • City - The city you want to fetch data from

GET data/{data point}/from/{country}

Get a specific type of data from a specific country

Requirements:

  • Data point - What kind of data you want to get
  • Country - The country you want to fetch data from



POST data/from/{city}/{data}

Post a specific type of data from a specific city

Requirements:

  • City - The city you want to post data from
  • Data - The data you want to post. It needs to follow the form of JSON

POST data/from/{country}/{data}

Post a specific type of data from a specific city

Requirements:

  • Country - The country you want to post data from
  • Data - The data you want to post. It needs to follow the form of JSON. The data needs to specify which city each data point comes from
